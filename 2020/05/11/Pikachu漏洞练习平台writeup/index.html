xss之htmlspecialchars

htmlspecialchars()是PHP提供的一个对特殊字符进行转义的函数，它可以把预定义的字符转换为HTML实体
预定义的字符：

&转换为&amp

“转换为&quot

‘转换为&#039

<转换为&lt

>转换为&gt

语法：htmlspecialchars(string,flags,character-set,double_encode)

在此主要关注第二个参数： 

可选。规定如何处理引号、无效的编码以及使用哪种文档类型。
可用的引号类型：
ENT_COMPAT - 默认。仅编码双引号。
ENT_QUOTES - 编码双引号和单引号。
ENT_NOQUOTES - 不编码任何引号。

getimagesize

getimagesize — 取得图像大小
语法：getimagesize ( string $filename [, array &$imageinfo ] ) : array

getimagesize() 函数将测定任何 GIF，JPG，PNG，SWF，SWC，PSD，TIFF，BMP，IFF，JP2，JPX，JB2，JPC，XBM 或 WBMP 图像文件的大小并返回图像的尺寸以及文件类型和一个可以用于普通 HTML 文件中 IMG 标记中的 height/width 文本字符串。

如果不能访问 filename 指定的图像或者其不是有效的图像，getimagesize() 将返回 FALSE 并产生一条 E_WARNING 级的错误。

使用cmd命令制作图片马：copy a.jpg + eval.php x.jpg
