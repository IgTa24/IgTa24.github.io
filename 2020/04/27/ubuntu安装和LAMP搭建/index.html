安装ubuntu

1.在官网下载镜像：

2.在VMware中创建新的虚拟机进行安装，不在此赘述

安装VMware Tools工具

安装好ubuntu后发现未铺满全屏，调整分辨率后也达不到全屏效果
于是我尝试重装VMware Tools
1.如图，点击重新安装

2.点击后桌面出现光盘文件，打开后将VMware Tools的压缩包提取到任意目录

我将其复制并解压在桌面新创建的文件夹中

3.打开解压后的文件，在该目录下打开终端

4.执行以下命令
sudo ./vmware-install.pl


一路默认敲回车即可

更新国内源

图形化界面设置

1.进入设置界面

2.选择About->Software Updates

3.选择Download from->other

选择国内源
4.点击重新载入后等待一段时间即可

在终端操作更新国内源

1.备份软件源文件
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
2.进入/etc/apt/目录，使用文本编辑器打开sources.list，注释原有内容，添加软件源

3.执行以下命令
sudo apt update 刷新软件源
sudo apt upgrade 更新软件
4.完成更新（图形化界面真香）

安装文本编辑器vim

sudo apt-get vim

安装Apache2

执行以下命令，安装完成后浏览localhost可看到Apache默认页面
sudo apt-get install apache2

启动/停止/查看服务
service apache2 start/stop/status
web内容默认在/var/www/html中
配置文件：
/etc/apache2/apache2.conf (主配置文件)
/etc/apache2/conf-* (其余配置文件)
/etc/apache2/sites-* (虚拟主机配置文件)

安装mysql

1.执行以下命令
sudo apt install mysql-server -y
sudo apt install mysql-client

2.尝试登陆mysql，报错。。。

3.进入/etc/mysql目录，获取debian.cnf的内容
sudo cat debian.cnf

获取密码
4.使用debian-sys-maint登陆
mysql -u debian-sys-maint -p
密码为debian.cnf中的password
5.修改root密码
ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'new password';
6.exit退出，完成

配置文件：
/etc/mysql/mysql.conf.d/mysqld.cnf
常用配置项：
bind-address 设置mysql接受请求的ip

安装php

1.安装php和对应mysql、apache模块
sudo apt-get install php php-mysql libapache2-mod-php
2.打开/etc/apache2/mods-enabled/dir.conf文件并将其更改为首先列出index.php

3.重启Apache服务器
systemctl restart apache2
4.测试php工作是否正常
创建php文件
sudo vim /var/www/html/1.php

保存后打开浏览器访问127.0.0.1/1.php

看到php信息页面即为成功

使用 php –ini 查看配置文件位置
默认：/etc/php/版本/运行模式/php.ini

搭建SSH服务

安装ssh
sudo apt install ssh
启动/停止/查看服务
service ssh start/stop/status
SSH配置文件默认存放在：/etc/ssh/sshd_config
